Información sobre Categorías

Las categorías organizan los productos de forma jerárquica (hasta 5 niveles). Las palabras clave de cada categoría se utilizan para la clasificación automática mediante análisis semántico. El sistema de IA aprende de estas palabras clave para categorizar nuevos productos de manera inteligente.
