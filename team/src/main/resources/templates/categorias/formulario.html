Nueva Categoría

Registra una nueva categoría para el sistema de clasificación inteligente

Sistema de Clasificación Semántica

Las categorías organizan los productos jerárquicamente y permiten la clasificación automática mediante IA. Las palabras clave que definas se utilizarán para identificar y categorizar productos de forma inteligente, ahorrando tiempo en el registro de tickets y mejorando la precisión del análisis de gastos.

Clasificación Inteligente

El sistema utiliza análisis semántico con IA para clasificar automáticamente productos basándose en las palabras clave que definas.

Importante

Puedes editar las categorías en cualquier momento. Las categorías inactivas no se usarán para clasificación automática pero se conservarán en el sistema.
