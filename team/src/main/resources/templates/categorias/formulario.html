Nueva Categoría

Registra una nueva categoría para clasificación de productos

Información sobre Categorías

Las categorías permiten organizar los productos y realizar clasificación automática de viáticos. Puedes crear categorías principales y subcategorías jerárquicas. Las palabras clave ayudarán al sistema de clasificación semántica a identificar productos de forma automática.

Información de la Categoría

Error en nombre
Nombre único e identificativo de la categoría
Selecciona una categoría padre para crear una subcategoría
Nivel jerárquico (1 = raíz, 2-5 = subcategorías)
Solo las categorías activas se usarán para clasificación
Descripción opcional para documentar el propósito de la categoría
Importante: Separa las palabras clave con comas. Incluye sinónimos, variaciones y términos relacionados. Estas palabras se usarán para la clasificación automática de productos.
Ejemplos de Palabras Clave:
Alimentos:

comida, alimento, comestible, food, eat, desayuno, lunch, cena

Bebidas:

bebida, drink, agua, refresco, jugo, café, té, water, soda

Transporte:

transporte, taxi, uber, bus, avión, flight, viaje, gasolina, combustible

Clasificación Inteligente

El sistema utiliza las palabras clave para realizar clasificación semántica automática de productos en tickets de viáticos.

Nota Importante

Una vez creada la categoría, puedes editarla en cualquier momento. Las categorías inactivas no se usarán para clasificación.
